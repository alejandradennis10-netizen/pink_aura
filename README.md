# pink_aura
La cosmetología y el maquillaje representan para mí una forma de combinar creatividad, arte y cuidado personal. Me apasiona aprender sobre técnicas de maquillaje, el cuidado de la piel, las tendencias de belleza y los productos cosméticos que ayudan a resaltar la belleza natural de cada persona.
